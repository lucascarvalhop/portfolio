Hotel Preamar

Hotel Preamar é um site que fiz para simular um site de um hotel praiano. O site foi feito no modelo One Page, uma página com um layout de rolagem longa onde o menu pode ser usado para navegar em cada seção do site. A criação do layout foi feita através da visita de vários sites de hoteis onde pude notar alguns "padrões" e então desenvolver um layout que casasse com o tema.

Tecnologias utilizadas

HTML / CSS / JavaScript
